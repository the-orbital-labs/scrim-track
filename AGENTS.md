# Instructions for Codex

Build this as a Chrome Extension Manifest V3 project.

Stack:
- TypeScript
- React
- Vite
- Tailwind CSS
- chrome.storage.local
- date-fns

Rules:
- Keep data local-first.
- Track only Scrimba URLs.
- Do not add a backend.
- Do not request broad permissions.
- Use small reusable functions.
- Prefer simple code over clever abstractions.
- After each change, run typecheck/build if available.
